# Quick Setup Guide

## For Tomorrow's Presentation

### Option 1: Quick Start (Recommended for Demo)
1. Open `index.html` in a web browser
2. The app will work, but the 3D map requires a Mapbox token

### Option 2: Full Setup with 3D Maps
1. Get a free Mapbox token:
   - Go to https://account.mapbox.com/
   - Sign up (free account works)
   - Copy your access token
   - Replace the token in `app.js` (line 5)

2. Run the application:
   ```bash
   npm install
   npm start
   ```

## Demo Flow

1. Click "Request a Demo" → Enter any email/password → Sign In
2. Click "Explore Our Solutions" → Opens 3D map
3. Select organization (e.g., "Redefine Retail")
4. Select space (e.g., "Centurion Mall")
5. Map will fly to location with 3D buildings
